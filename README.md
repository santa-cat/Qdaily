# Qdaily
简单模仿好奇心日报，主界面和设置页面<br/><br/><br/>

*主界面，循环播放viewgroup，支持前后滑动。<br/>
*主界面，下拉刷新，可以自定义下拉刷新头部。<br/>
*设置界面，模仿好奇心的跳转动画。<br/>
